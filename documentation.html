Documentation

Complete guide to using Backtrader Pro for developing and testing trading strategies.

Overview

Backtrader Pro is a powerful platform designed for traders and quantitative analysts to develop, test, and optimize trading strategies. Built with performance and flexibility in mind, it provides all the tools you need to create robust trading systems.

Key Capabilities

  • Strategy development with intuitive API
  • Historical backtesting with realistic market simulation
  • Parameter optimization and walk-forward analysis
  • Comprehensive performance analytics
  • Multiple data source integration
  • Custom indicator development

Core Concepts

Cerebro Engine

The Cerebro engine is the core of Backtrader Pro. It orchestrates all components including strategies, data feeds, brokers, and analyzers.

Strategies

Strategies contain your trading logic - when to buy, when to sell, and how to manage positions. They operate on data feeds and place orders through the broker.

Data Feeds

Data feeds provide market data to your strategies. They can come from various sources including CSV files, databases, or live APIs.

Broker

The broker simulates order execution, manages your cash and positions, and applies commissions and slippage.

Indicators

Indicators are mathematical calculations on price data (moving averages, RSI, etc.) that help inform trading decisions.

Creating Strategies

Strategy Structure

A basic strategy consists of:

  • __init__: Initialize indicators and parameters
  • next: Called for each bar of data, contains your trading logic
  • notify_order: Receive notifications about order status
  • notify_trade: Receive notifications about completed trades

Example Strategy Pattern

Create a strategy by defining your logic in the strategy editor:

  1. Initialize your indicators in the setup phase
  2. Define entry conditions (when to buy)
  3. Define exit conditions (when to sell)
  4. Configure position sizing and risk management

Order Types

  • Market Orders: Execute at the next available price
  • Limit Orders: Execute only at specified price or better
  • Stop Orders: Trigger when price reaches stop level
  • Stop-Limit Orders: Combination of stop and limit

Indicators

Built-in Indicators

Backtrader Pro includes a comprehensive library of technical indicators:

Trend Indicators

  • Simple Moving Average (SMA)
  • Exponential Moving Average (EMA)
  • Moving Average Convergence Divergence (MACD)
  • Average Directional Index (ADX)

Momentum Indicators

  • Relative Strength Index (RSI)
  • Stochastic Oscillator
  • Commodity Channel Index (CCI)
  • Rate of Change (ROC)

Volatility Indicators

  • Bollinger Bands
  • Average True Range (ATR)
  • Standard Deviation

Volume Indicators

  • On-Balance Volume (OBV)
  • Volume Weighted Average Price (VWAP)
  • Money Flow Index (MFI)

Custom Indicators

Create your own custom indicators by extending the indicator base class. Custom indicators can combine existing indicators or implement entirely new calculations.

Data Feeds

Supported Data Sources

  • CSV Files: Import historical data from CSV files
  • APIs: Connect to Yahoo Finance, Alpha Vantage, and other providers
  • Databases: Load data from SQL databases
  • Live Feeds: Connect to real-time data sources (Pro feature)

Data Format

Data should include at minimum:

  • Datetime/timestamp
  • Open, High, Low, Close prices
  • Volume (optional but recommended)

Multiple Timeframes

Use multiple timeframes in the same strategy to analyze different time horizons simultaneously.

Backtesting

Running a Backtest

  1. Select your strategy
  2. Choose your data feed and date range
  3. Set initial capital and position sizing
  4. Configure commission and slippage
  5. Run the backtest
  6. Analyze results

Realistic Simulation

Backtrader Pro simulates real market conditions:

  • Commission: Configure realistic commission rates
  • Slippage: Simulate price slippage on order execution
  • Position Sizing: Fractional position sizing and margin requirements
  • Data Alignment: Proper handling of multiple timeframes and symbols

Avoiding Look-Ahead Bias

The platform prevents common pitfalls like look-ahead bias by ensuring indicators and strategies only access historical data available at each point in time.

Optimization

Parameter Optimization

Find optimal parameter values for your strategy:

  • Define parameter ranges to test
  • Select optimization metric (Sharpe ratio, total return, etc.)
  • Run optimization across parameter space
  • Review results and select best parameters

Optimization Methods

  • Grid Search: Test all combinations of parameters
  • Random Search: Sample random parameter combinations
  • Genetic Algorithm: Evolutionary optimization approach

Walk-Forward Analysis

Validate strategy robustness using walk-forward analysis:

  1. Optimize parameters on in-sample data
  2. Test on out-of-sample data
  3. Roll forward and repeat
  4. Evaluate combined results
Warning: Be careful of overfitting! Always validate on out-of-sample data.

Performance Analysis

Key Metrics

  • Total Return: Overall percentage gain/loss
  • Sharpe Ratio: Risk-adjusted return measure
  • Maximum Drawdown: Largest peak-to-trough decline
  • Win Rate: Percentage of profitable trades
  • Profit Factor: Gross profit / gross loss
  • Average Trade: Mean profit/loss per trade

Visualizations

View comprehensive charts and graphs:

  • Equity curve over time
  • Drawdown chart
  • Trade distribution histogram
  • Monthly/annual returns heatmap
  • Rolling performance metrics

Trade Analysis

Examine individual trades in detail:

  • Entry and exit points on price chart
  • Trade duration and P&L
  • Winning vs losing trade analysis
  • Trade distribution by various factors

Example Strategies

Moving Average Crossover

Classic trend-following strategy using two moving averages. Buy when fast MA crosses above slow MA, sell when it crosses below.

RSI Mean Reversion

Counter-trend strategy that buys oversold conditions (RSI < 30) and sells overbought conditions (RSI > 70).

Breakout Strategy

Momentum strategy that enters positions when price breaks out of a defined range or channel.

Multi-Timeframe Strategy

Advanced strategy using multiple timeframes to confirm trend direction before entering trades.

Note: Example strategies are included in the installation for learning purposes. Always backtest and validate before using in live trading.
