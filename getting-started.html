Getting Started with Backtrader Pro

Welcome to Backtrader Pro! This guide will help you get up and running quickly.

Installation

Windows

  1. Download BacktraderPro-Setup.exe from the downloads page
  2. Double-click the installer to begin installation
  3. Follow the installation wizard prompts
  4. Launch Backtrader Pro from the Start menu or desktop shortcut

macOS

  1. Download BacktraderPro.dmg from the downloads page
  2. Open the DMG file
  3. Drag Backtrader Pro to your Applications folder
  4. Launch from Applications or Spotlight
  5. If prompted about an unidentified developer, go to System Preferences → Security & Privacy and click "Open Anyway"

Linux

  1. Download backtrader-pro.AppImage from the downloads page
  2. Make it executable: chmod +x backtrader-pro.AppImage
  3. Run the application: ./backtrader-pro.AppImage

System Requirements

Minimum Requirements

  • OS: Windows 10 (64-bit), macOS 11.0, or Ubuntu 20.04+
  • Processor: Intel Core i3 or equivalent
  • RAM: 4 GB
  • Storage: 500 MB available space
  • Display: 1280x720 resolution

Recommended Requirements

  • OS: Windows 11 (64-bit), macOS 12.0+, or Ubuntu 22.04+
  • Processor: Intel Core i5/i7 or AMD Ryzen 5/7
  • RAM: 8 GB or more
  • Storage: 2 GB available space (for data storage)
  • Display: 1920x1080 resolution or higher

First Launch

When you first launch Backtrader Pro, you'll be greeted with the welcome screen. Here's what to do:

  1. Choose Your Workspace: Select a folder where your strategies and data will be stored
  2. Review Preferences: Configure basic settings like theme and default timeframes
  3. Optional Tutorial: Take the interactive tutorial to familiarize yourself with the interface
Note: You can always access the tutorial later from Help → Interactive Tutorial

Configuration

Setting Up Data Sources

To start backtesting, you'll need to configure your data sources:

  1. Go to File → Preferences → Data Sources
  2. Click Add Data Source
  3. Choose from:
    • CSV Files (local data)
    • API Connections (Yahoo Finance, Alpha Vantage, etc.)
    • Database Connections
  4. Enter your connection details or file paths
  5. Click Test Connection to verify
  6. Save your configuration

Basic Preferences

Customize Backtrader Pro to your liking:

  • Theme: Choose between Light, Dark, or Auto mode
  • Default Cash: Set your starting capital for backtests
  • Commission: Configure default commission rates
  • Timezone: Set your preferred timezone for data

Your First Strategy

Creating a Simple Moving Average Crossover Strategy

Let's create a basic strategy to get you started:

  1. Click File → New Strategy
  2. Name your strategy (e.g., "SMA Crossover")
  3. Select your template or start from scratch
  4. Add your strategy logic using the built-in editor
  5. Configure parameters like moving average periods
  6. Save your strategy

Running a Backtest

  1. Open your strategy
  2. Click Run Backtest in the toolbar
  3. Select your data source and date range
  4. Set initial capital and other parameters
  5. Click Start
  6. View results in the analysis panel
Tip: Use the built-in strategy templates to learn best practices and get started faster!
